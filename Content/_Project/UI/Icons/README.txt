ITEM ICONS — drop PNG's hier
============================

Zet hier PNG-bestanden neer (256x256 of 128x128, transparante achtergrond) met EXACT
deze bestandsnaam (kleine letters, .png). De inventory laadt ze automatisch bij het
opstarten van de game. Staat er geen PNG, dan tekent de game zelf een nette gekleurde
tegel met een simpel symbool — dus alles werkt ook zonder bestanden.

Bestandsnaam        ->  Welk item                          (Street Connect pack-icoon)
------------------------------------------------------------------------------------
cash.png            ->  briefgeld (Cash)                   = CASH
bank.png            ->  ATM / bank-saldo                   = BANK
weed.png            ->  droge toppen (Bud_*)               = WEED
weed_wet.png        ->  natte toppen (WetBud_*)            = WEED (of DRUGS)
baggie.png          ->  verpakte grammen (Bag_*)           = WEED / PACKAGING
joint.png           ->  joints (Joint_*)                   = DRUGS
seed.png            ->  zaden (Seed_*)                     = WEED / SHROOMS
packaging.png       ->  containers/bench (Cont_*, bench)   = PACKAGING
papers.png          ->  vloei (Papers_*)                   = DOCUMENTS
soil.png            ->  aarde (Soil_*)                     = SUPPLY
water.png           ->  waterfles (WaterBottle_*)          = WATER
pot.png             ->  kweekpot (Pot_*)                   = SUPPLY
rack.png            ->  droogrek (DryRack_*)               = REPAIRS / SUPPLY
lamp.png            ->  groeilamp (Lamp*)                  = BOOST / ENERGY
tent.png            ->  kweektent (Tent*)                  = GARAGE / STASH
spray.png           ->  spray/fertilizer (Spray/Fert/Pest) = TUNING / REPAIRS
furniture.png       ->  meubels en overige (rest)          = STASH / INVENTORY

Tip: gebruik bij voorkeur de WITTE/DEFAULT variant met transparante achtergrond; de
inventory zet er zelf een gekleurd randje per categorie omheen.
